Module voice

Voice is a system to make the chat of Minetest more like a voice.

The only function that should be called from the client is activate.

Functions

abstruse (message, rate)
Abstruses the given message. That means that parts of it will be blanked, based on the rate.

Parameters:

  • message The message to abstruse.
  • rate The rate at which to abstruse the message, a value between 0 and 1, with 1 being everything abstrused.

Returns:

    The abstrused message.
activate ()
Activates the voice system.
in_range (distance, range, line_of_sight)
Checks if the given distance is in the given range, considering the line of sight.

Parameters:

  • range The range.
  • line_of_sight If there is line of sight.

Returns:

    true if it is in range.

Muffles the given messages, meaning replaces everything with dots.

Parameters:

Callback for if a chat message is send.

Parameters:

  • name The name of the sending player.
  • message The message that is send.

Returns:

    true if the message has been handled and should not be send.
random (rate)
Gets a random chance based on the given rate.

Parameters:

  • rate The rate. A number between 0 and 1, with 1 being always true.

Returns:
